Instructions to use API

Create user

POST: /api/users { "name":"Herman", "email":"herman.castano@gmail.com", "password": "12345" }
Token is returned in header: x-auth-token

Get current user

GET: /api/users/me
Auth token is returned in header: x-auth-token

Authenticate user

POST: /api/auth { "email":"herman.castano@gmail.com", "password": "12345" }
Auth token is returned in body
